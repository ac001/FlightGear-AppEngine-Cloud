  • CVS Logs and file updates (processed local and pushed to cloud, soon on cron)
  • Pulls developers list from CVS logs
  • Mapservers imported
  • MP status update with some beautiful soup
  • Latest issues from fliightgear-bugs
  • Aircraft Details
